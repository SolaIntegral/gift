# GIFT - 健康ギフトアプリ

## 概要

GIFTは「大切な人へ健康を贈る」をコンセプトとした健康ギフトアプリです。温かみのあるオレンジ基調のデザインで、スマートフォン最適化された直感的なUI/UXを提供します。

## 特徴

### 🎨 デザインコンセプト
- **温かみのあるオレンジ基調**: 安心感と親しみやすさを演出
- **丸みのある形状**: 優しさと親近感を表現
- **清潔感のあるレイアウト**: 信頼性と専門性をアピール
- **スマートフォン最適化**: タップしやすいボタンサイズと直感的な操作性

### 🎁 主要機能

#### 1. ホームページ (`index.html`)
- ヒーローセクションとキャッチコピー
- サービス特徴の紹介
- 人気ギフトの表示
- 利用の流れの説明
- お客様の声
- フッターナビゲーション

#### 2. ギフト送信機能
- **決まっている場合** (`send_gifts_nonhelp.html`)
  - ギフト選択から受取人情報入力
  - 決済処理
  - 完了メッセージ

- **迷っている場合** (`send_gifts_help.html`)
  - 質問フォームによる最適なギフト提案
  - 受取人情報入力
  - 決済処理
  - 完了メッセージ

#### 3. マイページ (`mypage.html`)
- ユーザー情報の表示
- 最新の履歴表示
- おすすめギフトの提案
- 通知設定
- 設定・サポートへのアクセス

#### 4. 履歴管理 (`history_send.html`)
- 送信・受信履歴の表示
- 検索・フィルター機能
- 統計情報の表示
- CSVエクスポート機能

#### 5. プロフィール管理 (`profile.html`)
- 基本情報の編集
- パスワード変更
- 通知設定
- プライバシー設定
- アカウント管理（削除、データエクスポート）
- 連携アカウント管理

#### 6. Aboutページ (`about.html`)
- サービス概要
- メリットの説明
- 利用シーン
- 利用の流れ
- よくある質問
- お問い合わせ

## 技術仕様

### フロントエンド
- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなスタイリングとレスポンシブデザイン
- **JavaScript (ES6+)**: インタラクティブな機能と動的コンテンツ

### デザインシステム
- **カラーパレット**:
  - プライマリ: オレンジ系 (#FF6B35, #FF8C42)
  - セカンダリ: グレー系 (#6C757D, #ADB5BD)
  - アクセント: 緑系 (#28A745, #20C997)

- **タイポグラフィ**:
  - フォント: システムフォント（読みやすさ重視）
  - サイズ: 16px以上（アクセシビリティ対応）

- **コンポーネント**:
  - ボタン: 丸みのある形状、ホバー・タップアニメーション
  - カード: 影付き、角丸デザイン
  - フォーム: 直感的な入力フィールド
  - ナビゲーション: 画面下部固定のフッターナビ

### レスポンシブデザイン
- モバイルファーストアプローチ
- ブレークポイント: 768px, 1024px
- タッチフレンドリーなインターフェース

## セットアップ

### 前提条件
- モダンブラウザ（Chrome, Firefox, Safari, Edge）
- ローカルサーバー（推奨）

### インストール手順

1. リポジトリをクローン
```bash
git clone [repository-url]
cd gift
```

2. ローカルサーバーを起動
```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx http-server

# PHPの場合
php -S localhost:8000
```

3. ブラウザでアクセス
```
http://localhost:8000
```

## 主要機能の実装詳細

### 1. ギフト提案システム
- 質問フォームによる受取人情報収集
- 年齢、健康への関心事、予算に基づく提案ロジック
- 動的なギフト表示と選択機能

### 2. 履歴管理システム
- 送信・受信履歴の表示
- 検索・フィルター機能（ステータス、日付、キーワード）
- CSVエクスポート機能

### 3. ユーザー設定管理
- プロフィール情報の編集
- 通知設定の保存（localStorage使用）
- プライバシー設定の管理
- パスワード変更機能

### 4. レスポンシブナビゲーション
- 画面下部固定のフッターナビ
- アクティブ状態の表示
- スムーズなページ遷移
